# django-course
complete django course tutorial and practice

# Initial Setup
## Install virtual env
```
pip install virtualenv
```
## name virtual env
```
virtualenv env
```
## Start virtual env
```
env\scripts\activate
```
To deactivate env
>env\scipts\deactivate
## Install Django
```
pip install django
```
## Create initial file structure
```
django-admin startproject project .
```
## Run server
```
python manage.py runserver
```
